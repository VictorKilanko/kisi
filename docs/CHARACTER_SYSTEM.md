# CHARACTER_SYSTEM.md — Characters, Government & Story Engine (Phase 1)

How the Republic's cast works, plus the 12 demo characters for the first
build. **Every character below is invented DEMONSTRATION content**
(`isDemo: true`, visible "Sample content" badge) to be replaced with the real
flock's names, photos, and histories per `CONTENT_CHECKLIST.md`. No real
Nigerian politicians are referenced or imitated.

```
THE REPUBLIC OF KISI

├─ OPPOSITION
│   Leader of the Opposition + shadow portfolios

    Kisi Electoral Commission · Bureau of Egg Statistics · The Coop Times (free press)
```

### Fictional parties (all invented; never real-party names/colors/slogans)
- **PPP — Progressive Peckers Party** (governing): "Forward with Feed."
- **NGA — New Grain Alliance** (opposition): "Count Every Egg."
- **FFF — Free Feathers Front** (minor/idealist): "Perches for All."
- Independents.

### Standing political issues (satire fuel)
Breakfast scheduling reform · the National Feed Budget · nesting-box expansion
· coop security modernization · egg production targets vs welfare · clean
water access · sports funding · senior hen retirement benefits · solar-light
schedules · the perennial Best Perch dispute · youth (chick) representation.

## 4. Story arcs planned for launch content

1. **Chi-Chi's First Egg** (emotional flagship): 5 timeline beats + 2 articles
   + egg milestone + presidential congratulation.
2. **The Missing Breakfast Grain** (political satire): allegation → Coop Times
   investigation → Assembly hearing → Bright Feather's apology tour →
   reconciliation.
3. **Mama Gold's Retirement**: legislation debate → farewell lay →
   retirement ceremony → "senior hen life" ongoing column.
4. **Perch Jumping Championship**: fixtures → results → rivalry →
   award ceremony (Hall of Fame seed).
5. **Flu-Season Recovery** (welfare-forward): Dr. Featherwell bulletins →
   quarantine explainer (real biosecurity education woven in) → recovery
   celebration. Fact/fiction badges keep the education clearly factual.

Each arc demonstrates a different content mechanic (milestones, articles,
bills, matches, health updates) so Phase 2 templates are exercised end-to-end.

## 5. Character lifecycle rules

- New chicken = one content file + portrait; profile, directory card, filters,
  and relationship links generate automatically.
- Status transitions (active → recovering → active; active → retired; any →
  memorial) are timeline events with dignified, template-guided copy.
- Memorial pages: gentle, celebratory of the life; no sponsorship button; no
  satire.
- Real-flock onboarding: when owner supplies real names/photos, demo
  characters are either retired from view or explicitly relabeled if the owner
  wants to keep a persona for a real bird — never silently blended.
